# Multi-Screen Sticky Notes App

## 📌 Overview
This application was created to help Customer Service teams manage notes more efficiently.
Instead of using physical sticky notes, this app provides a digital solution stretched across multiple monitors.

## 🚀 Features
- Multi-monitor support (stretched window across screens)
- Sticky notes creation and editing
- Auto-save to %AppData%
- Notes persist after restart
- Simple and fast UI for operators

## 💡 Problem Solved
Customer Service staff were juggling multiple paper notes, which caused:
- Lost information
- Inefficiency
- Messy workspaces

This app replaces that with a centralized digital board.

## 🛠️ Tech
- VB.NET / WPF (or WinForms — adjust)
- Local JSON storage
- Windows environment

## 📂 Storage
Notes are saved locally in:
%AppData%\YourAppName\notes.json

## ▶️ How to Run
1. Download repo
2. Open solution in Visual Studio
3. Build & run
